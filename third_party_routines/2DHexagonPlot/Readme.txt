Here is how to run the python script.

(1) Lines 13 and 14, you can set the row and column of the grid. 
please note that here 
M = the number of hexagons of each column;
N = the number of hexagons of each row.

(2) Lines 18-20 set the color of the hexagon.

(3) Line 23 set the path of the csv data file.

(4) run the python script by with "python 2DHexagonPlot.py"
