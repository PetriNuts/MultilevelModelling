Here we describe how to run the 2D movie generation routine.

Preparation:
1. Install python from https://www.python.org/
2. Install FFmpeg from https://ffmpeg.org/
3. Set environment variables, and put the bin folder in the path. E.g., C:\ffmpeg\bin.

Run:
python 2D_diffusion.py

Note:
1. Modify the row and column of the 2D grid at Line 21
2. Modify the data file name and separator at Line 16
3. This routine will generate a MP4 and a set of PNG figures.
